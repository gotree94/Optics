# Optics


📋포함된 주요 내용
1. 프로젝트 개요 - 연구 분야 및 주요 목표
2. 연구 영역 - 레이저 광학, 열상 광학, 열관리, 시뮬레이션
3. 현재 진행 프로젝트 - 4개의 주요 프로젝트 상세 설명
4. 기술 스택 - 사용 소프트웨어 및 도구
5. 설치 가이드 - Python 환경 설정 및 필수 패키지
6. 프로젝트 구조 - 체계적인 디렉토리 구성
7. 연구 아이템 리스트 - 5개 카테고리, 10개 세부 연구과제
8. 성능 지표 - 각 시스템별 목표 및 달성 성능
9. 워크플로우 - Mermaid 다이어그램으로 시각화
10. 참고문헌 - 관련 서적, 표준, 온라인 자료

🎨 주요 기능
  * 배지(Badge) 추가로 전문성 강조
  * 목차(Table of Contents)로 쉬운 네비게이션
  * 코드 블록으로 기술 사양 명확히 표시
  * 표(Table)로 성능 지표 비교
  * Mermaid 다이어그램으로 프로세스 시각화
  * 이모지로 가독성 향상
